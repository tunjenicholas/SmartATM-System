# 🏧 Smart ATM System

## 📖 Overview

Smart ATM System is a Java console-based banking application that simulates core ATM operations including deposits, withdrawals, transfers, PIN management, and transaction history tracking.

The project was developed to demonstrate object-oriented programming principles, business logic implementation, and secure account operations.

---

## 🚀 Features

### Account Management

* Unique account generation
* PIN generation
* Account validation

### Banking Operations

* Deposit funds
* Withdraw funds
* Transfer money between accounts
* Change PIN

### Transaction Tracking

* Mini statement generation
* Transaction history storage
* Timestamped transaction records

### Security

* PIN verification
* Login attempt restrictions
* Input validation
